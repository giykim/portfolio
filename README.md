# [Giyoung Kim's Portfolio](https://giykim.github.io/portfolio)

A personal portfolio website showcasing my professional experience, projects, education, and technical skills. Built with React and deployed on GitHub Pages.

## Features

- **Responsive Design** - Optimized for desktop and mobile viewing
- **Smooth Animations** - Powered by Framer Motion for fluid transitions
- **Scrollable Sections** - Interactive scroll-based navigation within pages
- **Modern UI** - Clean, code-inspired aesthetic with syntax highlighting

## Pages

- [**Home**](https://giykim.github.io/portfolio) - Welcome page with quick navigation to all sections
- [**About**](https://giykim.github.io/portfolio/about) - Education, technical skills, publications, and contact info
- [**Experience**](https://giykim.github.io/portfolio/experience) - Professional work history at Amazon, Duke University, Hagerty, and Broadsword
- [**Projects**](https://giykim.github.io/portfolio/projects) - Personal and academic projects

## Tech Stack

- **Framework:** React 18
- **Routing:** React Router DOM
- **Animations:** Framer Motion
- **Icons:** Font Awesome
- **Styling:** CSS with custom theming
- **Deployment:** GitHub Pages via gh-pages

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm

### Installation

```bash
cd portfolio
npm install
```

### Development

```bash
npm start
```

Opens [http://localhost:3000](http://localhost:3000) in your browser.

### Build & Deploy

```bash
npm run deploy
```

Builds the app and deploys to GitHub Pages.
